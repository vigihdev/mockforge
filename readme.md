## Mockforge

- A tool for generating mock data for WordPress entities (posts, pages, custom post types, taxonomies, terms, users, comments) efficiently via the command line.
- Features
  - Generate mock data for posts, pages, custom post types, taxonomies, terms, users, comments.
  - Customize the number of entities to generate.
  - Specify the post type, taxonomy, or term to generate mock data for.
  - Randomize the generated data using fakerphp/faker.
  - Save the generated mock data to a CSV file.

#### Powered by Symfony Console & fakerphp/faker
